Prescripto is a full-stack appointment booking platform built using the **MERN Stack**.

It provides three separate roles ‚Äî **Admin**, **Doctor**, and **Patient** - each with its own dashboard, functionalities, and secure authentication flow.

The system streamlines doctor discovery, appointment scheduling, profile management, and administrative control through a clean, responsive interface.

## **User Roles & Functionalities**

## 1. Admin Panel

Admin has complete control over system operations.

### **Admin Features**

* View all doctor appointments
* Cancel any patient‚Äôs appointment
* Add new doctors with full details
* Update doctor availability (active/inactive)
* Access admin dashboard with key metrics

---

## 2. Doctor Panel

Doctors get platform tools to manage their appointments and profile.

### **Doctor Features**

* View appointments booked for them
* Mark appointments as **Completed** or **Cancelled**
* Edit profile (fees, address, availability, etc.)
* Access doctor dashboard with appointment metrics
* Login using doctor credentials

---

## 3. Patient / User Panel

Patients can browse doctors, book appointments, and manage their data.

### **Patient Features**

* Browse doctors with **specialty filter**
* Book appointments
* View booked appointments
* Cancel appointment
* Edit profile (photo, name, contact, etc.)
